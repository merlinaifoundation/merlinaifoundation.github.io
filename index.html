What is Merlin AI Foundation?

Merlin AI Foundation is dedicated to making advanced computer automation accessible to everyone. Our flagship Python SDK and Docker-based platform let you create, control, and scale multiple intelligent agents—no complex setup, no steep learning curve.

Why Merlin?

Effortless Integration
Install with a single command. Start automating in minutes.
Multi-Agent Architecture
Launch and manage as many agents as you need—each with its own role, memory, and capabilities.
Docker-Powered Scalability
Deploy agents in isolated, reproducible Docker containers. Run locally or in the cloud with zero hassle.
Open Source & Extensible
100% open source. Build your own agents, extend functionality, and contribute to the future of AI automation.

Key Features

Simple Python SDK
Import, instantiate, and command agents with just a few lines of code.
Multiple Agents, Infinite Possibilities
Create specialized agents for different tasks—data processing, web automation, system monitoring, and more.
Seamless Docker Integration
Each agent runs in its own Docker container for maximum security and reproducibility. One command to build, launch, and manage your agent fleet.
Cross-Platform
Works on Windows, macOS, and Linux.
Developer Friendly
Comprehensive documentation, active community, and real-world examples to get you started fast.
from cua_sdk import ComputerAgent
agent = ComputerAgent()
agent.run_task("Open Excel and generate a report")
