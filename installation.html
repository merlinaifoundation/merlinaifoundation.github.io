Installation Guide

Python Version

Merlin AI Foundation requires Python 3.8 or higher. We recommend using the latest stable version of Python.

System Requirements

  • Operating System: Linux, macOS, Windows
  • Minimum RAM: 8GB
  • Recommended CPU: Multi-core processor

Install via Pip


# Basic installation
pip install merlin-ai

# Install with optional dependencies
pip install merlin-ai[extras]

Docker Installation


# Pull the latest Merlin AI image
docker pull merlinaifoundation/core

# Run a Merlin AI container
docker run -it merlinaifoundation/core

Install from Source


# Clone the repository
git clone https://github.com/merlinaifoundation/core.git

# Navigate to directory
cd merlin-ai

# Install dependencies
pip install -r requirements.txt

# Install the package
python setup.py install
